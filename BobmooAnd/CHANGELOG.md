# 변경 기록 (Changelog)

이 프로젝트의 모든 변경사항은 이 파일에 기록됩니다.

## [1.1.0] - 2025-12-06

### Changed
- UI 전면 개편
  - 새롭고 깔끔한 디자인으로 전체 화면 리뉴얼
  - 사용자 경험(UX) 개선

## [1.0.2] - 2025-11-18

### fixed
- 식단 데이터가 없는 날(휴일 등) 위젯이 갱신되지 않고 이전 데이터가 남아있던 버그 수정
- 데이터 부재 시 위젯에 '식단 정보 없음(텅...)' 상태가 올바르게 표시되도록 개선

### supported
- 빌드 경고 해결을 위해 Android 타겟 Java 버전 상향 (Java 17)

## [1.0.1] - 2025-10-25

### supported
- Google Play의 16KB 메모리 페이지 크기 요구사항을 충족하기 위해
  isar -> isar_community 포크 버전으로 교체

## [1.0.0] - 2025-10-07

### Added
- 정식 버전 출시 (v1.0.0)

## [0.1.4] - 2025-09-30

### fixed
- 업데이트, 재부팅시 생길 수 있는 버그 수정
- 설정에서 대표 식당 목록 API와 동일하게 -> 기본값도 일치시키기 위함
- 5x2 위젯에서 가로 사이즈(4x2로)를 줄이지 못하도록 수정
- 메뉴코스 개수에 따라 줄수 변경
    1. 1개 -> 3줄
    2. 2개 -> 2줄
    3. 3개 이상 -> 1줄

## refact
- WidgetUI 로직 파일 분리

## [0.1.3] - 2025-09-29

### Added
- 인앱 업데이트 알람 구현
- 백그라운드 데이터 동기화 로직 구현(for 위젯)
- 2x2 위젯 추가 (식당 하나의 식단을 보여주는 위젯)
- 5x2 위젯 추가 (모든 식당의 식단을 보여주는 위젯)
- 백그라운드 작업 추가 (6시간마다 식단 정보를 불러온다)

### fixed
- 운영 상태 표시 버그 수정
